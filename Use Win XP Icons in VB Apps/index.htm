Use Win XP Icons in VB Apps (VOTE HERE)

Intro:

I needed to use windows xp (32-bit alpha icons) icons in my vb app, but vb does not supports them. one day I found a example on vbaccelerator, the vbaccelerator example was fully working but I found one problem with it (not a bug).

the example shown at vbaccelerator needed we should make a Resource Script, in other words the resource file should be manually made.

like:

// Icons for this application
AAA ICON MOVEABLE PRELOAD 244.ico

and then needed to be complied using the rc.exe resource complier. although all this techniques works fine but some times its too much work or other issues/problem.

Issues/Problems:

Most vb app need to use several file for GUI, like bitmaps, audio, cursors and custom data. some developer including me put them in the app dir and some use the resource file made by vb resource editor.

But, If you modify or add resources in the resource file using the vb6 resource editor, it will remove your xp icon. and neither it appears in the editor.

In short you can't use both. if you want xp icon you can't add/modify the resource file and vise-versa.

Required Tools:

  1. Resource Hacker (free)
    search for "resource hacker 3.4"

Steps:

  1. Include the mod32BitIcon.bas in your project
  2. Add
    Call mod32BitIcon.SetIcon(Me.hWnd, "AAA")
    to the form load event
  3. make the project
  4. open resource hacker and open the exe you just build
  5. from the 'actions' menu select 'add new resource'
  6. open your xp icon and fill the text box shown below
  7. click 'Add Resource' and 'save' from the file menu.
  8. That All,

this will allow you to use xp style icons in you vb app for:

  1. without using resource at all
    or
  2. without breaking you existing resource file
